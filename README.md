# Stock.io – Backend

Este repositório contém o backend do Stock.io, um site de vendas fictício desenvolvido como projeto em grupo durante o período trainee na Empresa Júnior de Computação da UnB (CJR).  
O objetivo do projeto foi, após o grupo passar uma capacitação na área, aplicar na prática conceitos de desenvolvimento backend, trabalho em equipe, versionamento de código e organização.

## Descrição do Projeto

O backend do Stock.io é responsável por gerenciar a trasmissão de dados e a integração com o front-end da aplicação.  
Ele fornece uma API que permite a manipulação de informações relacionadas ao funcionamento de um site de vendas, como produtos, usuários categorias, items e operações básicas do sistema.


## Principais Funcionalidades

- Estruturação de uma API para um sistema de vendas
- Criação e organização de rotas
- Manipulação e validação de dados
- Integração com o front-end


## Tecnologias Utilizadas

- Node.js
- NestJS
- TypeScript
- Git e GitHub
